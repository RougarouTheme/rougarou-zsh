## Install
* Download the `rougarou.zsh-theme` file.
* Move rougarou.zsh-theme file to oh-my-zsh's custom themes folder: `~/.oh-my-zsh/custom/themes/`.
* Change `ZSH_THEME="rougarou"` the in `~/.zshrc`.
